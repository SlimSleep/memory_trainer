# Memory Trainer

Тренажёр памяти на Pygame с поддержкой локализации, пользовательских настроек и фоновой музыки.

## Структура проекта

- `main.py` - точка входа, инициализация приложения
- `config.py` - глобальные настройки окна, UI и аудио
- `localization/` - модуль локализации
  - `localization.py` - класс `Localizer` для перевода текста
  - `ru.json`, `en.json` - словари локалей
- `modules/audio.py` - обработка звука: фоновой музыки и эффектов
- `modules/ui/` - UI компоненты
  - `button.py` - класс `Button`
  - `label.py` - класс `Label`
  - `textbox.py` - класс `TextBox`
  - `slider.py` - класс `Slider`
  - `screen_manager.py` - класс `ScreenManager`
- `modules/screen/` - экраны приложения
  - `menu.py` - главное меню
  - `settings.py` - настройки со слайдерами для музыки и эффектов
  - `login.py` - экран входа
  - `match_pairs.py` - экран игры "Найди пару"
  - `sequence.py` - экран игры "Запомни последовательность"
- `modules/games/` - логика игр
- `assets/` - ресурсы
  - `sounds/` - аудио файлы, включая `background.mp3`
- `data/` - данные приложения
- `database/` - управление БД

## Особенности

- Отдельные ползунки для:
  - звуковых эффектов
  - фоновой музыки
- Фоновая музыка зацикливается из `assets/sounds/background.mp3`
- Переключение языка между русским и английским
- Настройки сложности, языка и уровня громкости в экране настроек

## Запуск

1. Активируйте виртуальное окружение:
   ```bash
   source .venv/bin/activate
   ```
2. Запустите приложение:
   ```bash
   python main.py
   ```

## Локализация

Приложение поддерживает русский и английский языки. Переключение доступно в настройках.
