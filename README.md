# Memory Trainer

Тренажёр памяти на Pygame с поддержкой локализации.

## Структура проекта

- `main.py` - точка входа, инициализация приложения
- `config.py` - настройки окна и FPS
- `localization/` - модуль локализации
  - `localization.py` - класс Localizer для перевода текста
- `modules/` - основные модули
  - `ui/` - UI компоненты
    - `button.py` - класс Button (кнопка с текстом и изображениями)
    - `label.py` - класс Label (текстовая метка)
    - `textbox.py` - класс TextBox (поле ввода текста)
    - `slider.py` - класс Slider (ползунок для выбора значения)
    - `screen_manager.py` - класс ScreenManager (управление экранами)
  - `screen/` - экраны приложения
    - `menu.py` - класс MenuScreen (главное меню)
  - `games/` - игры-тренажёры
- `utils/` - утилиты
- `assets/` - ресурсы (изображения, шрифты)
- `data/` - данные (сохранения, конфиги)
- `database/` - база данных
- `audio/` - аудио файлы

## Запуск

1. Активируйте виртуальное окружение: `source .venv/bin/activate`
2. Запустите: `python main.py`

## Локализация

Приложение поддерживает русский и английский языки. Переключение в настройках.
